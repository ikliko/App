# App
This is RestAPI with OAuth2 authentication. 

#### Requirements

- [x] php v7+
- [x] composer

#### Installation

Run following commands: 
- > composer install
- > Copy file ```.env.example``` and create ```.env``` file. Then Configure your database
- > php artisan migrate --seed
